# Andromeda

A comprehensive learning platform with backend API and mobile applications.

## Project Structure

```
andromeda-monorepo/
├── andromeda-backend/     # Node.js/Express API server
└── andromeda-mobile/     # Flutter mobile application
```

## Getting Started

### Backend

```bash
cd andromeda-backend
yarn install
yarn start
```

### Mobile App

```bash
cd andromeda-mobile
flutter pub get
flutter run
```

## Requirements

- Node.js 18+
- Flutter 3.x
- MongoDB
